# Zona21 - Atalhos de Teclado

Guia completo de todos os atalhos de teclado disponíveis no Zona21.

## Navegação Global

| Atalho | Ação |
|--------|------|
| `Cmd/Ctrl + 1-9` | Alternar para a tab na posição 1-9 |
| `Cmd/Ctrl + W` | Fechar tab ativa |
| `Cmd/Ctrl + Shift + ]` | Próxima tab |
| `Cmd/Ctrl + Shift + [` | Tab anterior |
| `Cmd/Ctrl + Shift + T` | Reabrir última tab fechada |
| `Cmd/Ctrl + \` | Alternar menu esquerdo |
| `Cmd/Ctrl + /` | Alternar menu direito |
| `?` | Abrir modal de atalhos |
| `Esc` | Fechar modais / Desselecionar asset |

## Home Tab (Biblioteca)

### Seleção e Navegação
| Atalho | Ação |
|--------|------|
| `Cmd/Ctrl + A` | Selecionar todos os assets |
| `Delete/Backspace` | Limpar seleção |
| `Enter` | Abrir asset selecionado no Viewer |
| `Space` | Próximo arquivo |
| `→` | Navegar para direita |
| `←` | Navegar para esquerda |
| `↑` | Navegar para cima |
| `↓` | Navegar para baixo |

### Marcação (Marking)
| Atalho | Ação |
|--------|------|
| `A` | Aprovar asset (toggle) |
| `D` | Rejeitar/Descartar asset (toggle) |
| `F` | Favoritar asset (toggle) |
| `Shift + A/D/F` | Marcar e avançar para próximo |
| `Cmd/Ctrl + Z` | Limpar marcação |

### Ações
| Atalho | Ação |
|--------|------|
| `Cmd/Ctrl + C` | Abrir Compare Mode (2-4 selecionados) |
| `Cmd/Ctrl + ,` | Abrir Preferências |
| `Shift + P` | Abrir Productivity Dashboard |

## Dicas

### Produtividade
- Use `Shift` + atalhos de marcação (A/D/F) para avançar automaticamente
- Use setas direcionais para navegação rápida no grid
- `Cmd/Ctrl + Shift + T` é útil para reabrir tabs fechadas acidentalmente
- Os menus laterais podem ser redimensionados arrastando a borda

### Marcação Eficiente
1. Use `A` para aprovar rapidamente as melhores fotos
2. Use `D` para rejeitar rapidamente as ruins
3. Use `F` para favoritar as excepcionais
4. Combine com `Shift` para marcação contínua

### Compare Mode
- Selecione 2-4 fotos e pressione `Cmd/Ctrl + C`
- Use `1-4` para selecionar o painel ativo
- Use `A/D/N` para marcar rapidamente
- Use `Space` para navegar pelos grupos

## Personalização

Os atalhos podem ser personalizados em:
- **macOS**: Preferências > Atalhos de Teclado
- **Windows/Linux**: Configurações > Atalhos de Teclado

Para personalizar, edite o arquivo:
```
~/.claude/keybindings.json
```

## Plataformas

### macOS
- Use `Cmd` como tecla modificadora principal
- `Option` pode ser usado em alguns contextos

### Windows/Linux
- Use `Ctrl` como tecla modificadora principal
- `Alt` pode ser usado em alguns contextos

## Acessibilidade

Todos os atalhos são otimizados para:
- Navegação por teclado completa
- Compatibilidade com leitores de tela
- Suporte a teclados internacionais
- Modo de alto contraste
